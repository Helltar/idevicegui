Installation
------------

Dependencies:

**Arch Linux**:

```
sudo pacman -S ifuse qt5pas
```

**Fedora**:

```
sudo dnf install ifuse usbmuxd qt5pas
```

**Ubuntu**:

```
sudo apt install ifuse usbmuxd libimobiledevice6 libimobiledevice-utils libqt5pas1
```

Done, unzip and run:

```
tar -xvf idevicegui-1.1.0.tar.gz && ./idevicegui
```
